## 🇧🇷 Sobre este Repositório
Este repositório centraliza o material de apoio e os desafios práticos sobre os **Princípios SOLID**, um dos pilares essenciais do Design de Código e da Engenharia de Software Orientada a Objetos. O objetivo deste módulo é aprender como criar softwares mais coesos, menos acoplados, fáceis de manter e de estender.

*   **Guia Teórico:** Explicação detalhada de cada princípio (SRP, OCP, LSP, ISP, DIP) diretamente nesta `README.md`.

*   **Laboratório Prático:** Códigos com violações arquiteturais estruturadas para você identificar, analisar e refatorar aplicando os conceitos aprendidos.

## 🇺🇸 About this Repository
This repository centralizes the supporting materials and practical challenges regarding the **SOLID Principles**, one of the essential pillars of Object-Oriented Software Design and Software Engineering. The goal of this module is to learn how to build software that is more cohesive, loosely coupled, and easy to maintain and extend.

### What you will find here:
*   **Theoretical Guide:** Detailed explanation of each principle (SRP, OCP, LSP, ISP, DIP) directly in this `README.md`.
*   **Lecture Slides:** Integrated presentation via GitHub Pages (see the link in the repository description).
*   **Practical Lab:** Source code containing structured architectural violations for you to identify, analyze, and refactor using the concepts learned.
